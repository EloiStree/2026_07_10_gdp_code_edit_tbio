# 2026_07_10_gdp_code_edit_tbio
> Simple user moddable Code Edit UI that can emit text, bytes, and IIDs (encoded as bytes).

TBIO concept is to make tool that received Text Byte and send Text Byte only as interface. 
It allows to make connection with Client Server Websocket.
